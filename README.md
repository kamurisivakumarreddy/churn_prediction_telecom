# churn_prediction_telecom
Predicting telecom customer churn using Python, EDA, and Machine Learning
Customer churn is one of the biggest challenges in the telecom industry.
Losing a customer is far more expensive than retaining one — yet most
companies only realize a customer is leaving after it's too late.

I built this project to answer a simple question:
"Can we predict which customers are about to leave — before they do?"

Using a real-world telecom dataset of 7000+ customers, I explored the
data, cleaned it, and trained machine learning models to predict churn.
The best model (Logistic Regression) achieved 80%+ accuracy.
